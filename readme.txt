ありがとうございます。

【注意】
　　　　このソフトウェアはファン作品として作られ、
　　　　イラストの著作権はすべて春画式様（@reply0715）に帰属します。


【このソフトウェアとは】
　　　　エロゲーっぽく画像を表示するだけのものです。
　　　　特になんかすごい機能とかはないです。


【動作環境】
　　　　Java Runtime Environment 8以上が必要となります。
　　　　Windows7のみでテストしました。LinuxやMacも多分動きます。

　　　　ぱぱっと作ったため描画は適当に済ませました。
　　　　サブのノートPC（Celeron3855U 2C2T 1.6GHz）だと結構重いので、
　　　　動作が重い方は画質を下げてください。
　　　　今後余裕があったら快適に動くようにしたい…


【利用方法】
　　　　春画式様のツイッター（@reply0715）から
　　　　お好きなエロ画像を落としていただいて、
　　　　image/0〜7/の中に画像を入れてください。

　　　　画像形式：　　*.png / *.jpg
　　　　ファイル名：　00.png　〜　99.png
　　　　　　　　　　　00.jpg　〜　99.jpg　（最大100枚）
　　　　pngとjpgは混ぜても多分問題ないはず（例：00.png、01.jpg…）
